# Skill Router

Inspect Skills exposed in the current session. Read the selected Skill's complete `SKILL.md` before invocation. Call only when its trigger matches and record trigger, inputs, outputs, status, limitations and evidence.

Statuses: `invoked`, `skipped`, `unavailable`, `failed`, `no_matching_skill`.

Essential unavailable capability blocks the promised gate. Do not claim a Skill was used merely because its name appears in a plan.
