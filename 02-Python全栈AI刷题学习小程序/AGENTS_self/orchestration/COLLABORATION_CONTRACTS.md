# Collaboration Contracts

- Independent discovery proposals precede cross-review.
- A veto contains owner, requirement, evidence, impact, minimum fix, verification and severity.
- A handoff contains objective, versioned artifacts, decisions, exclusions, verification, risks, blocker, allowed Actions, receiver and next gate.
- Contract, architecture, design-source, event-schema, provider, privacy, hosting or metric changes require an ADR.
- Rework contains expected, actual, evidence, allowed change scope, owner and re-verification; stop after the configured limit.
- Completion claims include environment, commands/steps, fresh evidence, uncovered scope, risks and real-vs-proxy distinction.
- Never log secrets, personal data, private user content or hidden reasoning.
