# 02-Python全栈AI刷题学习小程序

## 项目目标

做一个用于 AI 辅助学习和刷题的小程序项目，覆盖关卡、题目、答题、判分、用户进度、AI 出题和错题优化等核心流程。

## 技术栈

- 后端：Python 3.11+，FastAPI 或 Flask
- 前端：微信小程序
- 数据库：MySQL 或 PostgreSQL
- 缓存：Redis（可选）
- AI：大模型 API，用于生成题目、解释答案和优化错题

## 配置说明

真实密钥只能放在本地 `.env` 或部署平台 Secrets 中，不能提交到 GitHub。

需要准备的配置：

- `DATABASE_URL`
- `REDIS_URL`
- `AI_API_KEY`
- `AI_BASE_URL`
- `AI_MODEL`
