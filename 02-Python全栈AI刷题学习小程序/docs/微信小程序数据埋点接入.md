# 微信小程序数据埋点接入

## 查看入口

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)。
2. 选择本项目小程序。
3. 进入 `统计 -> 数据分析` 查看访问、用户、页面和来源等微信自动统计数据。
4. 进入 `统计 -> 数据分析 -> 自定义分析 -> 事件管理`，按下表创建事件和字段。
5. 手机微信可搜索“小程序数据助手”，查看被授权小程序的基础数据。

开发者工具中的 H5 预览不会进入微信数据后台。自定义事件应使用体验版或正式版真机验证，数据可能存在处理延迟。

## 自定义事件字典

| 事件 ID | 用途 | 字段 |
|---|---|---|
| `login_result` | 微信登录结果 | `status`, `login_type`, `duration_ms` |
| `search_toggle` | 联网搜索开关 | `enabled`, `scene` |
| `asset_select` | 选择文件或图片 | `asset_type`, `asset_count` |
| `knowledge_create` | 创建知识库 | `status`, `input_type`, `search_enabled`, `file_count`, `image_count`, `duration_ms`, `material_count`, `source_count` |
| `knowledge_supplement` | 补充知识库 | `status`, `search_enabled`, `file_count`, `image_count`, `duration_ms`, `material_count` |
| `quiz_start` | 开始闯关 | `question_count`, `search_enabled` |
| `answer_submit` | 提交答案 | `question_index`, `question_type`, `difficulty`, `is_correct` |
| `quiz_complete` | 完成闯关 | `question_count`, `correct_count`, `search_enabled` |
| `report_view` | 查看报告 | `source`, `score`, `total`, `mastery`, `search_enabled` |
| `report_share` | 分享报告 | `channel`, `has_report` |

事件 ID 和字段名必须与代码完全一致。建议先配置事件，再上传体验版进行真机测试。

## 推荐看板

- 知识库创建成功率：`knowledge_create.status = success / start`
- 联网搜索使用率：`knowledge_create.search_enabled = 1`
- 闯关完成率：`quiz_complete / quiz_start`
- 平均正确率：`answer_submit.is_correct` 的平均值
- 报告查看率：`report_view / quiz_complete`
- 分享率：`report_share / report_view`

## 隐私边界

埋点仅上报状态、数量、耗时和类别。禁止增加学习原文、题目内容、文件名、网页 URL、手机号、OpenID、登录凭证或 API Key。

代码入口：`frontend/src/services/analyticsService.ts`。非微信环境只输出开发日志，不调用微信上报接口。
