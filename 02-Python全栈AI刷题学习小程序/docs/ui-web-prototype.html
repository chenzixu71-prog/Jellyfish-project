端口在网络通信中主要负责什么？
A区分同一台电脑上的不同服务
B保存所有用户数据
C替代数据库完成查询
答对了，水母点头！ 端口像房间号，帮助请求找到具体的软件服务。
端口知识拆解
结论 端口的作用是：在同一台电脑里，帮网络请求找到具体的软件服务。
答错时怎么纠正 如果选了数据库或 Redis，说明你把“入口编号”和“服务本身”混在一起了。端口是入口，数据库和 Redis 是入口后面的服务。
为什么是这样
  • IP 地址负责找到哪一台电脑。
  • 端口负责找到这台电脑里的哪一个程序。
  • 不同服务通常监听不同端口，避免请求走错地方。
生活类比 一栋楼是服务器，房间号是端口。你先找到这栋楼，再根据房间号找到具体房间。
例子
  • 80：常见网页服务入口。
  • 3306：常见 MySQL 数据库入口。
  • 6379：常见 Redis 服务入口。
记忆口诀：IP 找电脑，端口找服务。
下一步练习 看到一个地址时，先问自己两个问题：它要访问哪台机器？它要进入哪个服务？
