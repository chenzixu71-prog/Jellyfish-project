# PRD Review Report: PRD_DRAFT_V0.md

## Review Scope

- PRD：`docs/PRD_DRAFT_V0.md`
- Handoff：`agent-runs/20260714-0001-discovery-geo-mvp/decisions.md`
- Required lenses：用户研究、产品、设计、前端、后端、测试
- 方法：五个角色独立评审，产品经理按共同根因合并，不重复计数。
- 事实与假设：GATE-1 决策、论文和竞品证据视为已接受输入；ICP 认可、建议价值和付费意愿仍是 Pilot 假设。

## Findings

### 阻断 1. 状态、部分成功与下游资格没有形成确定状态机

- 视角：设计 / 前端 / 后端 / 测试
- 位置：第 7、8、9、10、12 节
- 证据来源：PRD text + reviewer inference
- 问题：项目阶段与 Run 状态混合；取消、重试、部分成功、优化确认及正式对比资格未定义。
- 影响：设计无法画全状态，研发无法设置页面守卫，测试无法判断通过或失败。
- 建议：拆分工作流阶段、运行状态和优化草稿状态；规定部分成功门槛、取消终态、建议确认和可比条件。

### 阻断 2. 同条件比较缺少可验证条件指纹

- 视角：前端 / 后端 / 测试
- 位置：REQ-008、指标口径
- 证据来源：PDR fact + PRD omission
- 问题：遗漏模型参数、评测模板版本、问题顺序和 Mock 版本；时间被错误混入“同条件”概念。
- 影响：正文之外的变化可能被错误归因为优化效果。
- 建议：冻结 Evaluation Condition Snapshot；只有正文版本允许不同，时间只展示不参与相等判断。

### 阻断 3. Rule Score 与 15% 成功指标没有测试预言机

- 视角：用户研究 / 后端 / 测试
- 位置：第 3、11 节、REQ-006
- 证据来源：PRD text
- 问题：缺少维度、权重、缺失处理、舍入和相对提升公式；同一规则生成建议再自评存在循环。
- 影响：核心指标无法实现、复算或作为可信价值证据。
- 建议：锁定 `rules-v1` 评分契约；15% 仅统计有效 Real 对比的相对提升，同时与用户认可和建议质量一起判断。

### 阻断 4. Real Provider 凭证与失败责任边界未决定

- 视角：前端 / 后端 / 测试
- 位置：REQ-003、REQ-010
- 证据来源：PRD omission
- 问题：未决定用户自带 Key 还是服务端配置；失败分类和自动重试边界不足。
- 影响：UI、安全、部署和服务范围无法冻结。
- 建议：MVP 使用运营方服务端凭证，前端不接收 Key；仅临时失败可重试，Real 失败不得静默转 Mock。

### 阻断 5. Pilot 样本与逐需求验收不可计算

- 视角：用户研究 / 测试 / 产品
- 位置：第 3、4、13、15 节
- 证据来源：PDR fact + PRD omission
- 问题：有效用户、有效会话和排除规则缺失；REQ-001 至 REQ-010 无独立验收编号。
- 影响：无法证明目标用户价值，也无法建立端到端追踪矩阵。
- 建议：增加 Pilot 准入/排除/继续停止规则和 `AC-REQ-*` 验收表。

### 重要 6. 优化工作副本和建议冲突规则不足

- 视角：设计 / 前端 / 后端
- 位置：REQ-007
- 证据来源：reviewer inference
- 问题：多建议作用顺序、原文绑定、撤销、手改冲突和确认失效未定义。
- 影响：可能覆盖用户内容，Figma 和 Diff 行为不一致。
- 建议：建议只作用于工作副本；片段变化后需复核；支持撤销、重置和显式确认。

### 重要 7. 页面状态、响应式和中止反馈承接不足

- 视角：设计 / 前端
- 位置：第 8 节
- 证据来源：PRD text
- 问题：缺少运行上下文条、删除状态、中止反馈入口和窄屏策略。
- 影响：GATE-2 容易只覆盖理想路径。
- 建议：补页面状态矩阵；S03-S05 固定运行上下文条；1366px 基准，窄屏单列/标签页。

### 重要 8. 指标判定、N/A 和数据删除边界不稳定

- 视角：用户研究 / 后端 / 测试
- 位置：REQ-005、REQ-010、第 11-13 节
- 证据来源：PRD text
- 问题：品牌别名、Citation Availability、N/A 原因、删除范围和 30 天保留未成为可执行规则。
- 影响：指标不可重算，内容删除不可验收。
- 建议：补有限判定规则、统一原因码和最小数据清单；30 天作为 Pilot 固定默认值。
