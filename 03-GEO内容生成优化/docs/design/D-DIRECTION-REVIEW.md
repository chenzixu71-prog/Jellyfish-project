## What Changed

- 删除旧版黑色粗侧栏、荧光强调色和概念海报式标题。
- 使用浅色应用壳、克制钴蓝、Inter + Noto Sans SC 和 8px 间距体系。
- 将正文、证据和建议设为主视觉；GEO Rule Score 降为四项辅助指标之一。
- 使用项目面包屑、模型状态、视图 Tabs、文档工具栏和可撤销 change set 建立真实 Web SaaS 工作流。
- 建议面板区分高风险、中风险、已应用和无需修改，不为正向提示提供错误操作。
- 明确展示模型评测代理指标限制。
