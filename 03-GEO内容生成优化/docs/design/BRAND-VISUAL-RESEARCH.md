# Brand Visual Research

## 研究目标

为 GEO Insight Studio 找到一种可信、可扫描、有方法论辨识度的产品视觉，不复制竞品，也不落入“深色紫蓝 + 发光渐变 + AI 星星”的廉价模板。

## 参考与可借鉴部分

| 参考 | 可借鉴 | 明确不抄 |
|---|---|---|
| Linear 2026 UI refresh | 辅助导航降低对比度，核心任务区获得视觉权重；高密度但安静 | 深色灰蓝整体外观、过度依赖微弱边界 |
| IBM Carbon / IBM Design Language | 高密度 2x Grid、直接标注、概览后筛选再看细节、IBM Plex Sans Chinese | 企业蓝单色主导、复杂企业导航 |
| Vercel Geist | 文字层级、黑白对比、紧凑控件、精确像素感 | 黑白复制品、开发者官网式空旷大标题 |
| 生成式搜索竞品 | 需要持续展示平台/模型/问题集与运行元信息 | “大分数 + 一圈环图”替代证据、卡片海洋 |

## Mean Review：直接淘汰的视觉

1. **暗紫 AI 仪表盘**：行业同质化最严重，看不出 GEO 规则诊断，也削弱长文本可读性。
2. **Vercel 黑白复刻**：精致但没有项目身份，简历里只能说明会模仿，不能说明会建立品牌系统。
3. **满屏圆角卡片**：把工作流切碎，用户需要的是证据、建议和正文之间的关系，不是浏览组件展览。
4. **巨型总分**：Rule Score 是代理指标，视觉上不能压过原文证据和限制。
5. **假引用品牌墙**：Citation Availability 不等于语义核验，不用媒体 Logo 制造信任错觉。
6. **装饰性渐变球和玻璃拟态**：不服务判断，不进入产品。

## 三个候选方向

### A. Evidence Editorial / 证据编辑部

- 基调：纸白、墨黑、朱红、冷青，像编辑审校台而不是 SaaS 模板。
- 特征：文本批注、边栏索引、证据下划线、报告页有出版物质感。
- 优点：突出“改稿”和可解释证据，品牌辨识度高。
- 风险：做重会像新闻编辑器；数据区域需要更严谨的栅格。

### B. Signal Grid / 信号栅格（推荐）

- 基调：纯白、近黑、信号绿、警示珊瑚、少量高亮黄。
- 特征：2x 栅格、细线分区、直接标注、运行上下文条、像实验控制台但不黑客化。
- 优点：最适合高密度诊断、Before/After 和状态展示；开发可还原性最高。
- 风险：必须靠字体、品牌符号和生图素材避免过度工具化。

### C. Field Notes / 现场笔记

- 基调：冷灰白、深炭、湖绿、橘红，结合轻量纸张纹理和编号贴签。
- 特征：任务按“采样、诊断、修订、复评”呈现，过程感强。
- 优点：亲和，适合新用户理解复杂流程。
- 风险：贴签和纹理控制不好会显得手账化，不够 B2B。

## 推荐结论

优先深化 B，并从 A 借用“证据批注”和报告排版。C 只保留为空状态或 Pilot 教学素材的候选，不作为主系统。

## 资产策略

- 生图只生成品牌纹理、空状态插图、报告封面和证据/信号主题切图。
- 精确中文、指标、表格、按钮和表单全部在 Figma 中构建。
- 候选风格阶段先生成三方向情绪板；用户通过 GATE-2 后再按选定方向生成最终透明底素材包。
- Mobbin 连接器因付费限制不可用，已记录为 `unavailable`；不把不可访问内容当作证据。
