# AI 广告创意增长平台

面向广告投放和增长团队的 AI 产品原型，覆盖广告 Brief 输入、AI 多版本创意生成、A/B 测试、投放数据看板和 AI 自动复盘。

## 项目定位

这个项目不是单纯的“AI 文案生成器”，而是一个商业化广告优化闭环：

`广告 Brief -> AI 多版本素材 -> A/B 测试 -> 数据看板 -> AI 复盘 -> 下一轮优化建议`

适合写进简历的方向：

- AI 产品经理
- 商业化广告产品
- 增长产品
- 营销自动化
- 数据分析产品

## 功能

- 广告 Brief 创建：产品、行业、人群、卖点、平台、转化目标
- AI 创意生成：标题、主文案、CTA、短视频脚本、落地页首屏文案
- 多版本素材管理：A/B/C/D 版本、风格、人群、平台标签
- A/B 测试管理：实验假设、对照组、实验组、胜出版本
- 数据看板：曝光、点击、CTR、CPC、CVR、CPA、ROI
- AI 自动复盘：胜出原因、短板指标、下一轮测试建议
- 本地持久化：使用 localStorage 保存数据

## 如何运行

直接打开：

```text
index.html
```

也可以用任意静态服务器运行：

```bash
python -m http.server 8080
```

然后访问：

```text
http://localhost:8080
```
