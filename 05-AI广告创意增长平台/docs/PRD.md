# PRD：AI 广告创意增长平台

## 1. 背景

广告投放团队每天需要生产大量素材，并不断测试不同人群、卖点和表达方式。传统流程依赖人工经验，素材生产慢、复盘不系统、实验沉淀弱。

本产品用 AI 帮助投放团队完成从创意生成到投放复盘的闭环。

## 2. 目标用户

- 广告投放优化师
- 增长产品经理
- 中小商家
- 内容运营
- 品牌营销团队

## 3. 核心问题

- 不知道如何写高转化广告文案
- 素材版本少，测试速度慢
- 不知道哪个广告版本为什么表现好
- 投放数据分散，复盘依赖人工
- 下一轮优化方向不清晰

## 4. 产品目标

- 让用户 3 分钟内生成一组可测试广告素材
- 让用户能管理不同创意版本
- 让用户能配置简单 A/B 测试
- 让用户能看懂 CTR、CVR、CPA、ROI 等核心指标
- 让 AI 自动输出投放复盘和下一轮建议

## 5. 核心流程

1. 用户填写广告 Brief
2. 系统生成多版本广告创意
3. 用户选择版本进入实验
4. 系统模拟或接入投放数据
5. 看板展示关键指标
6. AI 生成复盘报告
7. 用户基于建议生成下一轮版本

## 6. MVP 范围

### 必须有

- Brief 表单
- 创意生成
- 创意版本卡片
- 实验管理
- 数据指标看板
- AI 复盘

### 暂不做

- 真实广告平台 API
- 多用户权限
- 支付
- 真实素材图片生成
- 企业级审批流

## 7. 关键指标

- Impression：曝光
- Click：点击
- CTR：点击率 = 点击 / 曝光
- CPC：点击成本 = 花费 / 点击
- Conversion：转化
- CVR：转化率 = 转化 / 点击
- CPA：单次转化成本 = 花费 / 转化
- ROI：投入产出比 = 收入 / 花费

## 8. 竞品参考
