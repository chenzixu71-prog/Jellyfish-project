# 01-短剧项目

这个目录用于记录 AI 短剧相关工作流、调研结果、角色资产规范、提示词模板和平台交付流程。

## 当前内容

- `ai-short-drama-studio/`: Codex skill，用于 AI 短剧项目从调研、剧本、人设、角色一致性、文生图、图生视频、配音字幕到交付 QA 的完整生产流程。

## 使用重点

- 真人风格短剧，而不是夸张动画风。
- 角色建模前后一致：脸型、年龄感、发型、服装、配饰、气质不能漂移。
- 表情需要丰富但克制：优先微表情、眼神、呼吸、嘴角、下颌、姿态变化，不使用过度夸张表演。
- 抖音/小红书调研只作为样本信号，不能把个别笔记当成平台整体结论。

## 后续记录建议

- `research/`: 抖音、小红书、红果短剧、可灵、即梦、剪映等调研记录。
- `projects/`: 每个短剧项目的立项包、剧本、分镜、提示词和成片 QA。
- `assets/`: 角色定妆图、表情图、场景图、封面图等资产索引。
