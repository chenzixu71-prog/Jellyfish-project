# AI VOC 与客户洞察回收

这是第 04 个项目目录，用来承接“AI VOC 与客户洞察回收”方向。

## 项目目标

围绕用户反馈、客服对话、评论、问卷和访谈记录，搭建一个可以帮助产品经理快速发现客户痛点、需求机会和优先级判断的 AI 产品。

## 计划闭环

```text
多渠道 VOC 数据 -> 清洗归类 -> 情绪 / 主题 / 高频问题分析 -> 洞察看板 -> 需求池 -> 产品改进建议
```

## 后续可建设模块

- VOC 数据导入：评论、客服记录、问卷、访谈纪要。
- 自动分类：问题、需求、抱怨、夸赞、流失风险、付费意向。
- 洞察看板：高频主题、情绪趋势、用户分层、需求热度。
- 需求回收：把用户反馈转成可评审的产品需求卡片。
- 产品建议：输出优先级、影响范围、验证方式和下一步动作。

## 当前状态

当前目录先作为项目入口占位，后续开发时会继续补充 PRD、设计稿、数据模型和 H5 Demo。
